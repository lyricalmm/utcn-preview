# UTCN Student Portal Preview

This is a prototype for a modern Student Portal for the Technical University of Cluj-Napoca (UTCN).

## Features

-   **Dashboard (`code.html`)**: Overview of classes, grades, and quick actions.
-   **Faculty (`facultate.html`)**: Detailed grade view, semester switching, and schedule integration.
-   **News (`stiri.html`)**: Latest announcements and academic news.
-   **Schedule (`orar.html`)**: Full weekly schedule with export options.
-   **Contracts (`contracte.html`)**: Study contract management and digital signing.
-   **Canteen (`cantina.html`)**: Daily menus for all campus locations.
-   **Facilities (`facilitati.html`)**: Digital student ID, library status, and dorm management.
-   **Payments (`plati.html`)**: Financial status and online payments.
-   **Store (`magazin.html`)**: Official university merchandise catalog.

## Technology Stack

-   **HTML5**
-   **Tailwind CSS** (via CDN)
-   **Google Fonts** (Inter)
-   **Material Icons**

## How to Run

Simply open `code.html` in your browser or serve the folder using a local server (e.g., `python -m http.server`).
